## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Edge, Firefox, Safari).
- (Optional) An [ElevenLabs](https://elevenlabs.io) API key for premium voice features.

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/bingo-tracker.git
   ```
2. **Navigate to the project**:
   ```bash
   cd bingo-tracker
   ```
3. **Launch the App**:
   Simply open `legacy/index.html` directly in your browser. No local server required!
